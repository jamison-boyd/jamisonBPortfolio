Program One

My Program One was converting a pre-existing program from a Struct to a C++ Class. The program held a movie database and the user was allowed to read in the movie data, save the data to files, manually add movies, delete movies, edit movie data, and display all the movies.

Program Two

Program Two was using Dynamic/Static Stacks and Queues to convert an Infix Notation Equation given by the user into a Postfix Notation, which is easier for the computer to read and solve the Equation. Only used the four main procedures, Addition, Subtraction, Multiplication, and Division.

Program Three

My Program Three assignment was using the same program for Program One; however, I implemented search and sorting algorithms and tested their times/efficiencies on different data sets.

There were only two Searching Algorithms, Linear and Binary. In contrast, I used six Sorting Algorithms, Bubble Sort, Insertion Sort, Insertion Descending Sort, Selection Sort, Quick Sort, and Merge Sort.

Program Four

Program Four came to me surprisingly easy compared to the first three. In this one, I created and implemented a Binary Search Tree to store data of customers of a restaurant, which allowed for easy searching, sorting, adding onto, and removing of data for the intended user.
